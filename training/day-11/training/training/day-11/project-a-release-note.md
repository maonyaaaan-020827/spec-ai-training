# Project A リリースノート

## プロジェクト名

[競合比較メモまとめツール]

## 公開URL

[[https://project-a-delta-nine.vercel.app/]](https://project-a-delta-nine.vercel.app/])

## 概要

[商談・提案の前に、散らばったメモから比較表と強み／弱点を数分で整理することができる]

## 主要機能

1. [他者との比較表を整理]
2. [自社の強み・注意点をまとめることができる]
3. [散らばったメモからでも整理することができる]

## 使用技術

- Next.js
- Tailwind CSS
- Vercel（ホスティング）
- AI（Cursor + Claude/ChatGPT/Gemini）

## 開発プロセスの振り返り

- かかった時間: 約5時間
- AIにどれくらい頼ったか: [エラーが起こった際にもAIに聞くのはもちろん、ターミナルでの答えが思い通りに進んでいるか逐一確認したので、利用頻度としてはかなり多かった]
- 自分で修正した箇所: [何回も作成し直したため、消したつもりのファイルが散乱しており、Git側がどのファイルを参照するかわからなくなっていたため、Geminiに聞きながらコードの修正を進めた]
- 苦労した点: [マニュアル通りに進めても、人の言葉や要件定義所の書き方で多種多様なためエラーが起きるのはつきもので、それを修正してまた実装しての繰り返しだと感じた。逐一不安な場合はAiに聞きながら進めると効率的だったのではないか、と思う]
