| **📖用語**  |  **✏️説明** |
|---|---|
|Token | AIが文章を処理する最小単位。一度の会話でどれくらい消費するかでプランが決まっている |
| Context（コンテキストウィンドウ） | AIの短期記憶で一度に覚えていられる限界値 |
| Prompt　 | AIへの指示・質問のこと。プロンプトの精度の高さでAIの解答の良し悪しが変わる |
| LLM | 大規模言語モデル。AIの頭脳そのものの技術 |
| LLM kinds | 頭脳のバリエーション。モデルの種類によって得意分野が異なる|
| AI Agent | 指示を待つだけでなく目的にために自律的にツールを使いこなす |
| Autonomous AI | 人間が細かく指示しなくてもゴールを示せば勝手に進めてくれるAI|
| RAG | AIが知らない社内資料や外部情報を参照し。回答する仕組み |
| OCR | 画像から文字計画を立ててから実行するモード |
| Skills | AIに（ブラウザ検索・計算機使用・画像生成などの特定の能力を追加する仕組み |
| AGENTS.md | AIエージェントにペルソナとルールを定義するための設定ファイル |
| CLAUDE.md | Claude Codeにプロジェクト固有のルール（マニュアルのようなもの）を伝えるための設定ファイル|
